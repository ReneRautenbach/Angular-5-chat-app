# About

This is a demo app that allows a user to sign in and enter a chat room.  


# Uses

 * HTML5 Websockets
 * [Express](https://github.com/expressjs/express)
 * [Babel](https://github.com/babel/babel-loader)
 * [Websocket/ws](https://github.com/websockets/ws)
